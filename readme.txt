Adds the official tweet button, to your site as a sidebar widget. You can use a twitter account for the entire site, or use personal accounts per author.


== Installation ==

This section describes how to install the plugin and get it working.

1. Install and activate the plugin in wordpress
2. Go to 'Widgets'
3. Put the Tweet Me widget somewhere you would like it to appear.
4. Give in the title for this widget
5. Typ the default twitter acount
6. Choose if it should use the default twitter account all the time, or if it should use the authors twitter account if possible,
